# Icon Placeholder

This folder should contain icon files for the application:

- `icon.png` - 512x512 PNG for Linux
- `icon.icns` - macOS icon file
- `icon.ico` - Windows icon file

You can create these from a base image using online tools or icon generators.

For now, the build will use default Electron icons.

To add custom icons:

1. Create a 512x512 pixel PNG with your app icon
2. Use a tool like <https://www.img2go.com/convert-to-ico> to create .ico
3. Use a tool like <https://cloudconvert.com/png-to-icns> to create .icns
4. Place all three files in this folder
5. Rebuild the app
